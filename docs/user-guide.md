# e-colleague ユーザーガイド

## はじめに

e-colleague は LINE グループ上で動作するタスク管理 AI アシスタントです。普段通りの会話の中から「やるべきこと」を見つけ出し、自動でタスクとして記録します。特別なコマンドを覚える必要はありません。

## 基本の流れ

```
会話する → タスクが生まれる → 自動で記録される
```

1. グループで普段通りに会話します
2. 「やる」「お願い」などの言葉をきっかけに、Bot がタスクを検出します
3. 「ここまでやればOK？」と聞かれたら、目安を答えるだけ（スキップも可）
4. タスクが自動で記録され、完了の目安も提案されます

## タスクを登録する

### 自然な会話からタスク化

一番の特徴は、**タスク登録のために特別な書き方をしなくていい**ことです。

```
A: スライド作らなきゃ
B: 明日までに
A: 俺がやる
```

この会話だけで、Bot が「スライド作成（期限: 明日、担当: Aさん）」というタスクを自動で記録します。

### ゴールの目安を伝える（任意）

タスクが検出されると、Bot がこう聞いてきます。

```
このタスクの「ここまでやればOK」という目安はありますか？
（なければAIが提案します）
```

- **目安を答える場合**: 「買って部室に置くまで」のように自由に回答
- **スキップする場合**: そのまま無視。AI が自動で目安を提案します

## タスクを確認する

### タスク一覧

グループで「**タスク一覧**」と送ると、未着手・進行中のタスクが一覧表示されます。

### メニュー

「**メニュー**」と送ると、よく使う操作のボタンが表示されます。

## タスクのステータスを変える

### ボタンで変更

タスク登録時に表示されるカードにあるボタンで、ステータスを変更できます。

- **進行中にする**: 作業を始めたらタップ
- **完了にする**: 作業が終わったらタップ
- **詳細を見る**: 実行ステップを表示

### チャットで完了報告

「ポスター発注終わった」「会場レイアウトやった」のように完了を伝えると、Bot が自動で該当タスクを完了にします。

## 使い方のコツ

- **「やる」「お願い」「私やります」が合図**: これらの言葉がトリガーになってタスク化されます
- **会話は自然に**: 「誰かポスター作って」「明日まで」「私やります」のように、普段の会話で十分です
- **ゴールの目安は気軽に**: 「買ってくるまで」「提出したらOK」くらいのゆるさで十分です

## できないこと

- 個人トークでは使えません（グループ専用です）
- タスクの編集・削除は LINE 上ではできません（管理者ポータルから行います）
- 過去の完了タスクを LINE 上で見ることはできません
